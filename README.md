# 시계열 예측 모델 성능 분석 보고서

## 1. 실험 개요

본 실험에서는 파동 데이터의 복잡도 증가에 따른 시계열 예측 모델의 학습 특성을 분석하고, 하이퍼파라미터 최적화 및 게이트 구조의 효과를 비교하였다. 또한 Teacher Forcing 비율 변화에 따른 장기 추론 성능과 누적 오차 양상을 분석하였다.

---

# 2. 파동 복잡도 증가에 따른 학습 특성 분석

파동의 중첩 횟수가 증가함에 따라 scatter plot이 데이터의 흐름을 제대로 따라가지 못하고 일정 값으로 수렴하는 현상이 관찰되었다. Training loss 그래프를 확인한 결과, number of waves가 1일 때는 loss 값이 0에 가까운 값으로 안정적으로 수렴하며 학습이 원활하게 진행되었다.

반면 wave의 개수가 2개 이상으로 증가하자 loss 값이 지속적으로 진동하며 수렴하지 못하는 현상이 나타났다. 이는 데이터의 비선형성이 증가함에 따라 초기 하이퍼파라미터 설정만으로는 복잡한 패턴을 충분히 학습하지 못했기 때문으로 분석하였다.

---

# 3. Optuna 기반 하이퍼파라미터 최적화

복잡한 시계열 패턴 학습 문제를 해결하기 위해 Optuna 라이브러리를 활용하여 하이퍼파라미터 최적화를 수행하였다.

최적화 대상 파라미터는 다음과 같다.

- Learning Rate
- Hidden Size
- Number of Layers
- Optimizer 종류

또한 각 파라미터와 MSE 값 사이의 상관관계를 분석하기 위해 파라미터별 성능 비교 그래프를 추가적으로 생성하였다.

총 15회의 반복 실험을 수행한 결과, 다음과 같은 최적 하이퍼파라미터 조합을 도출하였다.

- Learning Rate = 0.0005
- Number of Layers = 4
- Hidden Size = 128
- Optimizer = Adam

최적화 이전에는 MSE Loss가 1.0029 수준이었으나, 최적화 이후 0.2429까지 감소하는 성능 개선 효과를 확인하였다.

또한 최적 하이퍼파라미터를 적용한 상태에서 waves 개수를 3개로 증가시켜 학습을 진행한 결과, 기존 대비 prediction 결과가 데이터의 전체 흐름을 훨씬 안정적으로 추종하는 양상을 확인할 수 있었다.

---

# 4. 기울기 소실(Vanishing Gradient) 분석

기울기 소실 현상을 시각적으로 분석하기 위해, 가장 최근 시점을 x=0으로 설정하고 과거 시점으로 갈수록 x 값이 증가하는 역순 형태의 시각화 로직을 추가하였다.

실험 결과 input_len=30에서는 뚜렷한 기울기 소실 현상이 나타나지 않았으나 예측 성능은 최적 수준에 도달하지 못하였다. 반면 input_len=60에서는 일부 기울기 감소 현상이 존재했음에도 가장 우수한 예측 성능을 기록하였다.

이를 통해 최고 성능의 모델이 반드시 기울기 소실이 완전히 제거된 상태를 의미하지는 않으며, 일정 수준의 기울기 감소를 감수하더라도 충분한 과거 정보를 확보하는 것이 예측 성능 향상에 더욱 중요한 요소임을 확인하였다.

---

# 5. RNN, LSTM, GRU 구조 비교 분석

본 실험에서는 기본 RNN 구조와 함께 LSTM, GRU 모델의 성능을 비교 분석하였다.

LSTM과 GRU는 게이트 구조를 통해 장기 의존성을 효과적으로 유지하며 역전파 과정에서 발생하는 기울기 소실 문제를 완화하였다. 특히 게이트 구조는 모델의 기억 용량을 확장함으로써 불규칙파에 포함된 다중 주기성과 비선형 패턴을 동시에 학습할 수 있도록 한다.

실험 결과 LSTM이 가장 우수한 예측 성능을 기록하였으며, 이는 장기 기억 유지 능력이 강화되었기 때문으로 분석하였다.

반면 기본 RNN 구조는 상대적으로 짧은 기울기 소실 주기를 보이며 가장 낮은 성능을 기록하였다. 그러나 구조적으로 한계가 존재하는 기본 RNN 역시 하이퍼파라미터 최적화 이후에는 역전파 신호가 이전보다 안정적으로 유지되는 경향을 확인할 수 있었다.

따라서 모델 성능 향상에는 다음 두 가지 요소가 핵심적으로 작용한다고 판단하였다.

1. 게이트 구조 도입을 통한 정보 손실 방지
2. 하이퍼파라미터 최적화를 통한 학습 안정성 향상

# 6. 장기 추론(Long-term Forecasting) 누적 오차 분석

추론 과정에서 발생하는 누적 오차를 분석하기 위해 추론 길이를 10 step 단위로 최대 100 step까지 증가시키며 구간별 평균 MSE 오차를 측정하였다.

실험 결과 추론 길이가 증가할수록 오차가 점진적으로 누적되는 현상이 확인되었다. 특히 복잡한 비선형 시계열 데이터에서는 초기 예측 오차가 이후 단계로 전파되며 성능 저하를 유발하는 특징이 나타났다.

---

# 7. Teacher Forcing 비율에 따른 성능 비교

Teacher Forcing 비율 변화에 따른 학습 특성과 장기 추론 성능을 비교 분석하였다.

학습 단계의 Loss 곡선을 분석한 결과, Teacher Forcing 비율(TF)이 1.0일 때 가장 빠르고 안정적으로 수렴하며 최저 Loss 값을 기록하였다.

반면 TF=0.0 환경에서는 장기 의존성 학습에 취약한 기본 RNN 구조와 초기 하이퍼파라미터 한계로 인해 학습 수렴에 실패하는 현상이 나타났다. TF=0.5는 두 환경의 절충안으로서 중간 수준의 학습 안정성을 보였다.

장기 추론 단계에서는 10 step부터 100 step까지 누적 오차를 분석하였다. 오차 증가율은 TF=1.0, TF=0.5, TF=0.0 순으로 크게 나타났다.

특히 TF=1.0 조건에서는 초기 예측 오차는 매우 낮았으나, 추론 길이가 증가할수록 오차 증가폭이 커지는 현상이 나타났다. 이는 학습 과정에서 실제 정답 데이터에 지나치게 의존함으로써, 추론 단계에서는 스스로 발생한 오차를 교정하지 못하는 노출 편향(Exposure Bias) 현상 때문으로 분석하였다.

반면 TF=0.0 환경에서는 오차가 감소하는 양상이 일부 나타났으나, 이는 모델이 패턴 자체를 제대로 학습하지 못한 상태에서 발생한 결과로 판단하였다. 실제 예측 결과에서도 의미 있는 시계열 패턴을 형성하지 못하였다.

결론적으로 Teacher Forcing은 초기 학습 안정성 향상에는 효과적이지만, 장기 추론 상황에서는 노출 편향 문제를 유발할 수 있다. 따라서 데이터 복잡도와 모델 구조를 고려하여 적절한 Teacher Forcing 비율을 선택하거나, 학습이 진행됨에 따라 Teacher Forcing 비율을 점진적으로 감소시키는 전략이 효과적일 것으로 판단하였다.
